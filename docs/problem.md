1. 卡牌打出是点击触发，而实际的炉石传说游戏中，是将卡牌从手牌区拖出去，如果这张牌可以打出，脱离了手牌区后并松开左键便会打出去，如果玩家拖动时反悔了，又拖回手牌区，再松开该牌便会回到手牌中，费用也不会扣除
2. 现在战场上的卡牌布局是锁死位置的，而实际的炉石传说游戏，卡牌的位置是动态变化的。如空场时，放下一个随从，这个随从会自动处于站场中央，在他左边放一个随从，则两个随从靠着中轴线。而不是你现在这种卡牌要放到固定的位置。
3. 带有指向性（如选中一个生物攻击英雄或生物，会在七点到终点动态绘制一个箭头）
4. 手牌中费用够、可打出的牌，都会在轮廓显示一个绿色的光环。战场上能动的随从也是（如这回合可以攻击）其轮廓外也会显示一个绿色的光环。而可以被攻击的目标，当拖动箭头到该目标上时，目标轮廓会显示一圈红色的光环。
5. 手牌现在布局很糟糕，当手牌中牌不多时分散的太开了，应当呈现一个较为紧凑的扇形，我觉得这里你需要修改这个扇形分布函数，具体你可以参考一下炉石传说手牌的布局
