# 游戏内容
类似炉石传说一样的卡牌游戏，但是我们的游戏内容完全不同

基础卡牌种类：
1. 生物卡（也就是炉石传说中的随从卡），拥有基础攻击力与生命值，这种卡一般都会带有一些[特殊效果](#特殊效果)。
2. 帝王卡（类似炉石传说中的英雄卡，但不完全一致），打出帝王卡可以改变玩家的英雄单位，如本来我们的英雄单位为嬴政，在打出了汉武帝的英雄卡后英雄便化身为了汉武帝，技能也变成汉武帝的技能，文臣也切换为汉武帝时期的文臣。所以一张帝王卡还会伴随一些附属卡，如文臣（我们的游戏不像炉石传说有武器卡，我们是文臣卡，每个文臣拥有自己的技能，花费对应能量点即刻触发技能，每个文臣都会拥有一个技能那就是退场，这个技能花费一点能量点，让玩家可以切换到另一个文臣，切换的新文臣本回合不能使用技能。每张英雄卡能使用的文臣数量是有限的，一般是3-5个）
3. 将领卡，将领卡是高级的生物卡，除了生物卡的基础属性、效果外，每张将领卡都会带自己的技能（目前这个阶段，就按照炉石传说的泰坦卡设定来做将领卡就行）
4. 政令卡，本质就是炉石里圣骑士单位的光环卡，我们这里与其他法术卡区分开来。（例子：政令卡A：减少己方生物卡打出时消耗的能量点1点，持续3回合；政令卡B：每个回合结束己方场上的生物卡获得+1/+1）
5. 圣旨卡，本质就是炉石里的法术卡，但是一般都没有直接伤害效果（几个例子：圣旨卡A:打出时效果是己方每个场上生物获得效果：攻击时获得一点能量；圣旨卡B：打出时从牌库中召唤一个随机武将卡到己方场上）
6. 巫术卡，一种邪术的法术卡（即政令卡与圣旨卡），效果强大但也往往带有负面效果
7. 妙计卡：todo

# 游戏规则
目前来说与炉石传说相同，击杀敌方英雄即获得胜利。不过游戏不像炉石那样分法师、牧师等，而是玩家选择一个起始英雄（从固定的几个中选），然后与国家绑定，如起始选择了中国的帝王，那这 套牌就只能从中国的牌库以及中立牌库中选择。初步文明我们设置中国、日本、美国、英国和德国五个。

# 特殊效果
战吼、亡语、光环（如这个生物在场上时，增加己方单位一点攻击力）、嘲讽（这个生物在场上时，他不死地方生物不能攻击己方非嘲讽随从）、突袭（打出的回合就可以攻击地方生物）、刺客（打出的回合不可以攻击地方生物，但可以攻击敌方英雄）、冲锋（打出的回合可以攻击地方生物和敌方英雄）

# 游戏开发技术栈
ts，要支持多人游戏，主界面、卡牌收藏管理、套牌管理，原生支持多语言适配。游戏效果不能太死板，要注重细节的生动性。
